# alexruf.github.io

![Build](https://github.com/alexruf/alexruf.github.io/workflows/github%20pages/badge.svg)

Source for [alexruf.net](https://www.alexruf.net) — a personal blog about code, open source, and the things I build for fun.

Built with [Hugo](https://gohugo.io) and the [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme.

## Local Development

```bash
hugo server
```

## Update Theme

```bash
git submodule update --remote themes/PaperMod
```
